> The project is configured to use CUDA 13.0 with the following setup:
>

> 运行前先安装 pytorch
> 
> `uv pip install torch==2.9.0+cu130 torchvision==0.24.0+cu130 --index-url https://download.pytorch.org/whl/cu130`
